1. It is required to save the images in the same folder as the code and where you are executing from.
2. To change the input image, just change the lines 545 and 546 accordingly.
3. The output images get saved in the same folder where the code is being executed from.
4. If the input image is not BMP, you will recieve a warning in the beginning.
5. To compile the code, you can simply use g++ compiler i.e. simply g++ exp1_code.cpp
6. We have also included the executable, and to run it, simply type ./a.out
